Daily UI

This month, I decided to challenge myself by creating one UI feature a day. I signed up for the Daily Logo Challenge and the Daily UI Challenge. I received a logo and a UI feature prompt every day. I would sketch the logo out on my tablet, then create it in Adobe Illustrator. Then, I used that logo to inspire my UI design, which I created on Sketch.
