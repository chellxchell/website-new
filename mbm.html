A Private Psychiatry Practice

Mainline Behavioral Medicine is a project for a private psychiatry practice in Paoli, Pennsylvania. My goal was to make the site look professional, but still simple enough that users can find information easily. I started by making preliminary sketches of the layout for the interface - I wanted the website to be one page, so that all of the information is easily accessible.

Recognizable Branding

Logo
After deciding on a layout, I did some trial and error in Illustrator to create a logo. I finally decided on a lettermark logo in a dark blue color that appears white on the website for visibility. I wanted the logo to be simple but recognizable.

Colors
After deciding on a logo, I built the color scheme around the color I created the logo in. I wanted to color scheme to be professional, but still visually appealing. I added in lighter colors for highlighting items on the website.

Digital Prototype

Once I had a layout and a color scheme, I started building the site. I added a navigation bar that went to different sections of the one page website.




Moving Forward

I plan to also add a section in the website for patients to view necessary forms as pdfs. I am also waiting on the client to provide pictures for both of the physicians and a biography for one of them.
